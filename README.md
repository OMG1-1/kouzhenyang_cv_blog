## 关于本书

本项目旨在打造一本开源免费、新手友好的计算机视觉和自然语言处理入门教程。

- 全书采用图文形式，内容清晰易懂、学习曲线平滑，引导初学者探索计算机视觉和自然语言处理的知识地图。
- 鼓励读者互助学习，提问与评论通常可在两日内得到回复。

若本书对您有所帮助，请在页面右上角点个 Star :star: 支持一下，谢谢！
